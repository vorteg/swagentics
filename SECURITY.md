# Security Policy

## Supported Versions

Only the latest version of Swagentics is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v2.0.x  | :white_check_mark: |
| < v2.0  | :x:                |

## Reporting a Vulnerability

We take the security of our framework seriously. If you find a vulnerability, please do not report it via public issues.

Instead, please send an email to the repository owner or use the GitHub Private Vulnerability Reporting feature if enabled.

Please include:
- A description of the vulnerability.
- Steps to reproduce.
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for the fix.
